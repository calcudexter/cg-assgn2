## Key bindings :  
### Global rotation
1. w, s: Rotation about x-axis
2. a, d: Rotation about y-axis
3. q, e: Rotation about z-axis

### Bike model movement
1. f : front wheel movement
2. r : rear wheel movement
3. t : steering movement
4. SHIFT + <f/r/t> : Opposite movement

### Rider movement
1. Torso
2. Arms
3. Shoulders
4. Hips
5. Legs
6. Ankles
